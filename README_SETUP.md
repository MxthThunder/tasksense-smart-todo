# 🚀 QUICK START - Multi-Page Setup

## Run These Commands:

```bash
# 1. Install React Router
npm install react-router-dom

# 2. Run the dev server (after copying files)
npm run dev
```

## Copy These Files:

**Root folder has .txt files. Copy them to correct locations:**

1. `AddTaskPage.tsx.txt` → `src/pages/AddTaskPage.tsx`
2. `AddTaskPage.css.txt` → `src/pages/AddTaskPage.css`  
3. `TasksPage.tsx.txt` → `src/pages/TasksPage.tsx`
4. `TasksPage.css.txt` → `src/pages/TasksPage.css`
5. `EditTaskPage.tsx.txt` → `src/pages/EditTaskPage.tsx`
6. `NewApp.tsx.txt` → Replace entire content of `src/App.tsx`

## Result:

✅ **Page 1** (`/`) - Add new tasks
✅ **Page 2** (`/tasks`) - View all tasks  
✅ **Page 3** (`/edit`) - Edit tasks
✅ **Floating ➕ button** - Quick add from any page
